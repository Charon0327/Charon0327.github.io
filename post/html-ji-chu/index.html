HTML基础

2021-01-25 2 min read # HTML

HTML基本结构
<!doctype html>文档声明，此类为H5文档声明
< meta>包含了页面的隐式信息
某些元素具有空内容，空元素在开始标签内进行关闭。
HTML属性
属性总是在HTML元素的开始标签中设置
属性总是以名称/值对的形式出现
始终属性值加引号
常用属性：id class title style href src align
文本标签
h1-h6标签可定义标题。h1最大，h6最小
p标签定义段落
b标签规定粗体文本
u标签为文本添加下划线
s标签克定义加删除线文本定义
br可插入一个简单的换行符
hr>标签在HTML页面中创建一条水平线
i标签规定斜体文本
列表标签
ol定义有序列表，常用属性：start规定列表起始位置
ul定义无序列表
li定义列表项目
dl定义了定义列表，用于结合


dt定义了定义列表中的项目
dd在定义列表中定义条目的定义部分
快捷方式
ol>li{列表$}*4
超级链接
a来设置超文本链接
鼠标移动到链接上，鼠标默认会变为一只小手，链接文本默认带有下划线
herf属性指定跳转地址 href=""
target属性固打开方式 target='_blank'（新窗口下打开）
支持相对路径与绝对路径的跳转
链接跳转
相对路径：相对于当前文档路径进行跳转
绝对路径：从盘符开始的路径
往上一级查找方式../文件夹/文件名
同级查找方式：文件夹/文件名
本地锚点跳转：< a href="#id值"> < p id="值">< /p>
图片标签
< img src="地址"> 通过此标签，可向页面内引入图片
src属性为文件的路径，alt属性为图片由于某种原因无法展示的时候显示的信息
